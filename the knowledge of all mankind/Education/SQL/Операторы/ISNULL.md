## Описание
Оператор `ISNULL` позволяет узнать равно ли проверяемое значение `NULL`, т.е. пустое ли значение.

Для примера выведем всех преподавателей, у кого отсутствует отчество:
```sql
SELECT * FROM Teacher
WHERE middle_name IS NULL;
```
