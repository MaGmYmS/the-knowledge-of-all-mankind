## Описание
Для числовых литералов в SQL есть все привычные нам арифметические операторы:

| Оператор | Описание              | Пример        |
| :------: | :-------------------- | :------------ |
|  %, MOD  | Деление по модулю     | 11 % 5 = 1    |
|    *     | Умножение             | 10 * 16 = 160 |
|    +     | Сложение              | 98 + 2 = 100  |
|    -     | Вычитание             | 50 - 51 = -1  |
|    /     | Деление               | 1 / 2 = 0.5   |
|   DIV    | Целочисленное деление | 10 DIV 4 = 2  |

Пример:
```sql
SELECT (5 * 2 - 6) / 2 AS Result;
```
